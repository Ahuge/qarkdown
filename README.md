# qarkdown

A Qt [markdown](http://daringfireball.net/projects/markdown/) preview app.

## Dependencies

- Qt 4.7.4 an higher
- [sundown-0.1](https://github.com/sschober/sundown)

## Building

The usual

    qmake
    make

should do it.
